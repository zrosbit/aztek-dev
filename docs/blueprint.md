# **App Name**: Aztek Nexus

## Core Features:

- Aztek Connect CRM Hub: A centralized Partner OS that serves as the destination for leads generated from the smart quote engine, providing project lifecycle tracking and co-branded marketing assets.
- eWarrantyFy Automation Engine: Integrated document lifecycle automation that issues QR-secured performance certificates instantly and provides a public-facing verification portal for customer transparency.
- Smart Claim Review Tool: An AI-powered tool that uses reasoning to analyze claim evidence, installation photos, and historical job records to detect potential fraud or technical inconsistencies.
- Moto Precision-Cut Library: A searchable database of OEM-spec pre-cut patterns for 500+ bike models across 6 categories, enabling installers to select exact kits with 99.4% fit accuracy.
- Aztek Academy Tiered Certification: A modular learning and quality control gateway featuring 3 certification tiers across 5 discipline tracks that unlocks partner ecosystem access and warranty rights.
- Multi-Vertical Product Discovery Catalog: A managed digital catalog covering automotive, moto, and architectural verticals that drives both the public-facing product discovery platform and internal partner ordering.
- Evidence-Based Installer Mobile Interface: Mobile execution tool for installers to manage job cards, capture sign-off photos, and verify GPS-stamped checklists for quality assurance and warranty activation.
- Interactive Partner Locator & Appointment Booking: A consumer-facing map and booking system that allows customers to find certified partners and schedule installations based on real-time availability.
- Operational Analytics Forecasting Tool: An AI-powered tool that uses reasoning to process operational data and partner performance to forecast regional demand patterns and inventory consumption.
- Multi-Step Smart Quote Engine: A dynamic customer-facing tool for generating immediate project quotes across vehicle and architectural protection sectors, feeding qualified leads into the CRM.
- Integrated CMS & Resource Center: A content management module for partners and admins to manage technical blogs, maintenance guides, and product FAQs without developer intervention.

## Style Guidelines:

- Primary background: Obsidian Black (#0A0A0A) for a professional, software-forward technology aesthetic.
- Accent color: Electric Blue (#2563EB) used for primary CTAs and highlighting technological innovation.
- Surface hierarchy: Graphite (#1A1A1A) for navigation and Charcoal (#2A2A2A) for modular card components.
- Font pairing: 'Space Grotesk' for technical headers and 'Inter' for functional, high-legibility body text.
- Modular 'command-center' layout using subtle borders (rgba(255,255,255,0.08)) to organize data-dense operational zones.
- Minimalist linear icons utilizing semantic colors (Success Green #22C55E, Warning Amber #F59E0B, Danger Red #EF4444) for status signaling.
- Sophisticated micro-transitions with CSS blur effects to create a modern 'digital fog' loading experience.
